# fdf
fdf project

Objective was to crete basic rendering engine for height map.\
Using just listed below functions
new window
paint pixel
put string
put img
create img
create window
del img
del window
clear window

available projections
iso
iso 2:1
tow view
side view

My verison is able to perform basic transformations like
moving
zooming
changing altitude

usage
Because of the mlx fdf is only compatible with macOS(n and later).\
to try other maps clone repository, and run fdf
./fdf maps/map.fdf
clone and use make run
make run

lines are calculated with Brasenhems algorithm, but i ll soon rewrite it with wus.
